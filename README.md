# FinGuide IA

## Objetivo

Criar um assistente virtual capaz de orientar usuários sobre segurança digital utilizando Inteligência Artificial e uma base de conhecimento estruturada.

## Tecnologias

- Python
- IA Generativa
- Engenharia de Prompts
- Markdown
- GitHub

✔ Evita alucinações

✔ Indica quando não possui informação

✔ Sugere próximos passos
